# Product Definition

## Project Title
Mythrax Sidecar Intelligence Engine

## Vision Summary
Mythrax is a 100% native Rust local memory, model routing, and cognitive self-improvement engine designed for autonomous AI agents. It operates as a low-latency sidecar intelligence daemon that provides a single-port API gateway, a persistent database engine (SurrealDB/RocksDB), and a split GPU semaphore model broker. It is engineered to build high-cohesion project-level and global memory representations to eliminate forgetting loops and seamlessly integrate forged assets into an agent's active context window using resource-efficient local models.

## Core Objectives
1. **Short-Term Context Recall:** Provide immediate short-term retrieval for active agents operating with large context windows.
2. **Project-Level Memory:** Build high-cohesion, project-specific knowledge representations (clusters) for shared operational context.
3. **Global Wisdom:** Maintain a durable global partition for general guidelines and architectural rules.
4. **Forged Knowledge:** Enable raw reference assets (PDFs, specs) to be dynamically injected into active contexts via RAG.
5. **Resource Efficiency:** Optimize token footprint and compute overhead using local models and efficient text embeddings.

## Target Audience
Autonomous AI agents, specialized subagents, and human developers coordinating multi-agent systems via MCP integrations.
