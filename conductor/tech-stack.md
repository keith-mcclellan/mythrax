# Technology Stack

## Core Language & Runtime
- **Language:** Rust (100% native)
- **Async Runtime:** Tokio
- **Web/API Framework:** Axum

## Database & Persistence
- **Primary Database:** SurrealDB
- **Storage Engines:** SurrealKV / RocksDB

## AI & Machine Learning
- **Local Inference:** MLX (`mlx-rs`) for Apple Silicon, ONNX Runtime (`ort`)
- **Tokenization:** `tokenizers`

## Observability & Serialization
- **Logging:** `tracing`, `tracing-subscriber`
- **Serialization:** `serde`, `serde_json`, `serde_yaml`
